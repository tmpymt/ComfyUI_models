## 模型列表

### Flux Texture
- 文件名: Flux Texture.safetensors
- 大小: 约172MB
- 描述: 这是一个用于ComfyUI的纹理生成模型
- 由于文件大小超过GitHub的100MB限制，请从以下链接下载：
  - [Hugging Face](https://huggingface.co/tmpymt/ComfyUI_models)

## 使用方法
1. 下载模型文件
2. 将模型文件放置在ComfyUI的models目录下的适当位置
3. 重启ComfyUI
4. 在节点中使用相应的模型
